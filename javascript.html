Stock Price Tracker

    Developed a live stock trading app using Flask, enabling users to track stocks by ticker symbols, visualize 5-day historical data with Chart.js, and receive real-time price updates every 15 seconds. Implemented a responsive frontend with HTML, CSS, and JavaScript (jQuery), utilizing AJAX for data fetching and localStorage for persistence. The app supports easy addition and removal of stocks.
