Produkt Deployment med Kotlin

Langsigtet mål:

    Viden Færdigheder Kompetencer
    1. Implementering af brugergrænsefladeelementer i Kotlin.
    2. Implementering af layout elementer i Kotlin.
    3. Implementering af navigationsstruktur i Kotlin.
    4. Integration af Azure Storage SDK til håndtering af filopladninger og -downloads.
    5. Integration af fejlhåndtering i Kotlin for at håndtere netværksfejl og filkonflikter.
    6. Testning af appen på forskellige enheder og skærmstørrelser for at sikre en fejlfri brugeroplevelse.
    7. Konfiguration og udgivelse af appen på Google Play Store med passende beskrivelse, screenshots og app-ikon.
    1. Evnen til at kunne benytte sig af Kotlin kodesprog, og kunne programmere en bruger venlig grænseoverflade.
    2. Evne til at implementering bruger venligt layoutdesign.
    3. Evne til Kotlin's layout-API'er og biblioteker
    4. Færdigheder i tredjeparts-API-integration (Azure Storage) og evnen til at implementere sikker og pålidelig filbehandling i en mobilapp.
    5. Kendskab til Kotlin's fejlhåndteringsmekanismer og -strategier. Samdigt at kunne implementere robuste fejlhåndteringsløsninger.
    6. Kompetencer i mobilapp-testning, fejlfinding og forståelse af enheds- og skærmkompatibilitet med evnen til at løse brugeroplevelsesproblemer.
    7. Kompetencer i appkonfiguration, -udgivelse og forståelse af retningslinjer samt krav fra Google Play Store, kombineret med evnen til at skabe tiltalende og informative app-listinger.
    1. Evne til at programmere brugervenlige grænseflader ved hjælp af Kotlin og implementere layoutdesign.
    2. Færdigheder i at implementere brugervenligt layoutdesign.
    3. Færdigheder i at implementere brugervenligt layoutdesign ved hjælp af Kotlin's layout-API'er og biblioteker.
    4. Evne til at integrere tredjeparts-API'er som Azure Storage til sikker og pålidelig filbehandling i en mobilapp.
    5. Kendskab til Kotlin's fejlhåndteringsmekanismer og evne til at implementere robuste fejlhåndteringsløsninger.
    6. Kompetencer i mobilapp-testning, fejlfinding og forståelse af enheds- og skærmkompatibilitet for at løse brugeroplevelsesproblemer.
    7. Evne til at konfigurere og udgive apps på Google Play Store i overensstemmelse med retningslinjer og krav, samt evne til at skabe tiltalende og informative app-listinger.

Cloud Management med Azure

Langsigtet mål:

    Viden Færdigheder Kompetencer
    1. Azure Storage-tjenesten og dens anvendelsesmuligheder til filopladning og -downloads i Kotlin-apps.
    2. Viden om integrationen af Azure Storage SDK i Kotlin-appen til sikker og pålidelig filopladning til skyen.
    3. Azure Cost Manager til overvågning og styring af omkostningerne ved skyressourcer, inklusive forståelse af forbrugsmønstre og optimeringsmuligheder.
    4. Azure Role-Based Access Control (RBAC) og dets anvendelse i oprettelsen og styringen af brugergrupper samt tildeling af relevante tilladelser til Azure Storage-tjenester.
    5. Definition og implementering af regler og sikkerhedsforanstaltninger ved hjælp af Azure RBAC for at sikre autoriseret adgang til skyressourcer.
    6. Azure Cost Managers analyseværktøjer til evaluering af forbrug og omkostninger i forbindelse med Azure Storage-tjenester.
    7. Optimeringsmetoder og -værktøjer til finjustering af Azure Storage-løsningen for at opnå bedre ydeevne, skalerbarhed og omkostningseffektivitet.
    1. Forståelse af Azure Storage-tjenesten og dens anvendelsesmuligheder i forbindelse med filopladning og -downloads.
    2. Integrere Azure Storage SDK i Kotlin-appen for at muliggøre sikker og pålidelig filopladning til skyen.
    3. Azure Cost Manager til at overvåge og administrere omkostningerne ved skyressourcer, herunder forståelse af forbrugsmønstre og optimeringsmuligheder.
    4. Azure Role-Based Access Control (RBAC) og evnen til at oprette og administrere brugergrupper samt tildele relevante tilladelser til Azure Storage-tjenester.
    5. Definere og implementere regler og sikkerhedsforanstaltninger ved hjælp af Azure RBAC for at sikre, at kun autoriserede brugere har adgang til skyressourcerne.
    6. At anvende Azure Cost Manager til at analysere forbrug og omkostninger forbundet med Azure Storage-tjenester og træffe informerede beslutninger om ressourceallokering og -optimering.
    7. Optimere og finjustere Azure Storage-løsningen for at opnå bedre ydeevne, skalerbarhed og omkostningseffektivitet, herunder at anvende best practices og værktøjer til skyoptimering.
    1. Designe og implementere effektive databaseløsninger i Azure.
    2. Sikre og administrere Azure-tjenester på en effektiv måde.
    3. Optimering af omkostninger og forbrug i Azure for at maksimere effektiviteten.
    4. Træffe informerede beslutninger baseret på analyse om rettighed oprettelse for at oprette ønsket sikkerhedsniveau.
    5. Kunne oprette brugergrupperinger der kan tildeles rettigheder for at oprette rettigheder til en gruppe brugere med ønskede rettigheder.
    6. Identificere omkostninger Azure-miljøet for at sikre kontinuerlig drift og under et budget.
    7. Kunne optimere og oprette den rette type Azure Storage baseret på den ønskede lagrede data.
